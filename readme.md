# Yu-Ken-Po! (Yu-Gi-Oh | Jo-ken-po Edition)

## Sobre o projeto
Jogo de jokenpo criado acompanhando uma aula explicativa

### conceitos abordados na aula:
- Armazenamento e gerenciamento de estado manual
- Funções limpas
- Organização de código

## Detalhes Adicionados
- Ao jogo inicial foi adicionado o método "Melhor de 3" para que o jogo acabe e possa ser reiniciado ao completar 3 rounds com vitórias ou 3 rounds com derrotas;
- Somente as cartas sorteadas ao início podem ser usadas;
- Foi adicionado uma função para garantir que a carta escolhida pelo usuário e pelo computador seja removida das opções disponíveis;
- As cartas são continuamente entregues cartas até que o player ou o computador ganhe;
- Função para selecionar uma carta aleatória para o computador, a função é expansível para caso seja adicionado um método de escolher aleatóriamente para o player;
- Um contador para exibir também os rounds empatados;
- Também foi adicionado uma forma de mostrar o resultado do jogo na tela de confronto.
